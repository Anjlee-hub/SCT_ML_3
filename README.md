# Cat and Dog Image Classification Using SVM

## SkillCraft Technology - Machine Learning Task 03

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs.

## Technologies Used

- Python
- NumPy
- Matplotlib
- Pillow
- Scikit-learn
- Joblib
- Google Colab

## Machine Learning Algorithm

Support Vector Machine (SVM)

## Project Workflow

1. Loaded cat and dog images.
2. Resized images to 64 x 64 pixels.
3. Converted images into RGB format.
4. Flattened images into feature vectors.
5. Normalized pixel values between 0 and 1.
6. Split the dataset into training and testing sets.
7. Trained an SVM model using a linear kernel.
8. Predicted cat and dog classes.
9. Evaluated the model using accuracy, precision, recall, and F1-score.
10. Created a confusion matrix.
11. Visualized sample predictions.

## Classes

- 0 - Cat
- 1 - Dog

## Dataset

Cats and Dogs Mini Dataset from Kaggle containing 500 cat images and 500 dog images.
